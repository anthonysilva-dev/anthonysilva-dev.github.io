3. Glossário Técnico

HTML
Linguagem de marcação usada para estruturar páginas web.
Atualmente na versão HTML5, mantida pelo W3C.
CSS
Linguagem de estilos que define a aparência visual do HTML.
Permite criar layouts responsivos com media queries.
JS
JavaScript
Linguagem de programação executada nos navegadores, usada para deixar páginas interativas.
DOM
Representação em árvore do HTML que o JavaScript consegue manipular.
API
Interface que permite que dois sistemas se comuniquem.
HTTP
Protocolo de comunicação entre o navegador e o servidor.
URL
URI
Endereço que identifica um recurso na internet.
DNS
Sistema que transforma o nome de um site em endereço IP do servidor.
