# Projeto CSS Zen Garden

## Autor
Anthony da Silva Cavalcante

## Tema
Layout limpo com fundo claro e barra lateral escura

## Design de Referência
Mid Century Modern, por Andrew Lohman

## Link de Referência
https://www.csszengarden.com/221/

## Descrição
Usei CSS Grid para dividir a página em duas colunas: o conteúdo principal à esquerda e a sidebar à direita. O header usa Flexbox. Em telas menores o layout passa para uma coluna só.

## Recursos Utilizados
- CSS Grid
- Flexbox
- Variáveis CSS
- Media Queries
