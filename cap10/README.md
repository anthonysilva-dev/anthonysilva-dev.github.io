# Projeto — Recriação Responsiva

## Autor
Anthony da Silva Cavalcante

## Site Escolhido
Netflix

## Link do Site
https://netflix.com

## Objetivo
Recriar uma versão própria e responsiva da homepage da Netflix, com header, hero section, linhas de cards e seção de benefícios.

## Estratégia Responsiva
Mobile-first com Flexbox e CSS Grid, expandindo via min-width.

## Breakpoints
- 768px (tablet)
- 1024px (desktop)
- 1440px (tela grande)

## Recursos Utilizados
- CSS Grid
- Flexbox
- clamp()
- prefers-color-scheme
- Media Queries

## Adaptações Realizadas
Simplifiquei o layout para facilitar a responsividade. Os cards usam Grid com número de colunas variável por breakpoint. O nome do serviço foi alterado para StreamFlix por ser um projeto acadêmico.

## Dificuldades
Fazer a hero section funcionar bem em todos os tamanhos de tela mantendo a imagem de fundo legível.
